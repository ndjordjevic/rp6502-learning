## Overview

The FNIRSI DPS-150 is a compact, adjustable DC regulated power supply with precise voltage and current control. It operates in **step-down (buck) mode** -- the input voltage must always be higher than the output voltage.

- **Output voltage**: 0 - 30V (resolution 10 mV)
- **Output current**: 0 - 5A (resolution 1 mA)
- **Output power**: 0 - 150W
- **Input voltage**: DC 5.0V - 32V (supports PD and QC fast-charge protocols, power banks)
- **Display**: 2.8" color TFT (320x240)
- **Size**: 106 x 76 x 28 mm, ~178g
- **Output connectors**: 4.0 mm banana plug terminals (positive and negative)
- **Data interface**: Micro USB (PC software control and firmware upgrade)
- **Power input**: DC barrel jack and Type-C (do NOT use both simultaneously)

## Output modes

- **CV (Constant Voltage)**: Maintains set voltage, current varies with load.
- **CC (Constant Current)**: Maintains set current, voltage varies with load.

The display shows which mode is active (CV or CC).

## Protection mechanisms

| Protection | Abbreviation | Adjustable range |
|-----------|-------------|-----------------|
| Overvoltage | OVP | 0 - 30.00V |
| Overcurrent | OCP | 0 - 5.1A |
| Overpower | OPP | 0 - 150.00W |
| Overtemperature | OTP | 0 - 99C |
| Undervoltage (input) | LVP | 0 - 30V |
| Reverse input | REP | Built-in |
| Reverse output (back-feed) | -- | Built-in |

When any protection triggers, the device automatically shuts off output and sounds a beep alarm. Under LVP, output is prohibited until input voltage recovers.

|---------|------------|------------|
| **Power** button | Run/Pause (start/stop output) | Lock/unlock control panel |

| **Encoder** (scroll wheel) | Enter settings; cycle voltage/current selection | Switch main page / waveform display |
| **Left/Right** buttons | Main Interface: Switch Data Group<br>Waveform Display: Decrease/Increase Time Base<br>Settings: Left/Right navigation | Continuously cycle left/right |
| **Double-arrow** button | Main Interface: Highlight appears on page<br>Settings: Enter specific setting option | Switch between main page and waveform display |

### Parameter adjustment

1. Press the encoder to enter adjustment mode (voltage highlight appears).
2. Press again to cycle between voltage and current setting.
3. Scroll the wheel left/right to change the highlighted digit.

### Preset data groups

The device supports **6 preset groups** (M1-M6), each storing a voltage and current setting:

1. Short press the set button on the main page to enter preset interface.
2. Press encoder to select voltage/current, scroll to adjust.
3. Short press set button to save.
4. Use left/right buttons on the main page to switch between presets.

## Waveform display

The DPS-150 includes a real-time waveform display showing voltage and current over time. Access by long pressing the encoder from the main interface. You can zoom in/out when viewing a single waveform.

## PC software control

The DPS-150 can be controlled from a PC via Micro USB:

1. Connect device to PC/Mac with a Micro USB data cable.
2. Select the communication port (check Device Manager on Windows, or `/dev/tty.*` on Mac/Linux).
3. Click "Connect" in the software.

**Note**: When connected to PC software, the device buttons are locked.

### Official software (Windows/Linux only)

- **Download**: Available from FNIRSI's official downloads page: https://www.fnirsi.com/pages/software-downloads
- **Software name**: "FNIRSI Power Supply for PC (Windows/Linux)"

### Mac-compatible alternative (WebSerial)

For Mac users, there's a **web-based solution** using WebSerial API:

- **GitHub project**: https://github.com/cho45/fnirsi-dps-150
- **How it works**: JavaScript/WebSerial implementation that runs in **Google Chrome browser**
- **Setup**:
  1. Clone or download the repository
  2. Run a local web server: `npx serve .` (or any static file server)
  3. Open the local URL in **Google Chrome** (WebSerial API is Chrome-only)
  4. Connect via Micro USB and use the web interface

**Note**: WebSerial API is currently only supported in Chrome/Chromium browsers, not Safari or Firefox.

### Software features

- **Basic**: Set voltage/current, monitor output, control on/off.
- **Sequential output**: Program a sequence of voltage/current steps with delays and loop counts.
- **Current sweep**: Fixed voltage, sweep current through a range with configurable step and delay.
- **Voltage sweep**: Fixed current, sweep voltage through a range with configurable step and delay.
- **Waveform recording**: View, store, and delete waveform data (official software only).

## Firmware upgrade

**Method 1 (direct)**:
1. Download latest firmware from the official website.
2. Hold the set button while connecting to PC via Micro USB -- device enters upgrade mode and appears as a USB drive.
3. Copy firmware file to the USB drive.
4. Device upgrades automatically and restarts.

**Method 2 (via PC software)**:
1. Connect and pair with PC software.
2. Click "Firmware Upgrade" then "BOOT/Restart".
3. Copy firmware to the USB drive that appears.
4. Device upgrades and restarts.

## Safety and precautions

- **Step-down only**: Input voltage must be higher than output voltage.
- **Input range**: DC 5V - 30V. Below 5V = undervoltage lockout. Above 30V = device damage.
- **Do NOT power both input ports simultaneously** -- this can damage the lower-voltage source.
- **Inductive/capacitive loads**: Connect the load first, then enable output.
- **Battery charging**: Add a Schottky diode on the output to prevent reverse current.
- **Heat**: High-power output causes heating -- use in a well-ventilated area.

## Relevance to RP6502

The DPS-150 is your **current-limited bench power supply** for safely powering the RP6502 board during testing:

- **Pre-IC power test**: Set voltage to 3.3V with a low current limit (e.g., 100-200 mA) to test power rails for shorts before inserting ICs. If OCP triggers, there's a short on the board.
- **Controlled power-up**: After ICs are inserted, gradually increase current limit while monitoring for abnormal draw.
- **CV mode** for normal operation: Set 5V (USB-level) or whatever the board expects, with current limit set above normal operating current but below damage threshold.
- **CC mode** for fault diagnosis: Limit current to a safe value and observe voltage to identify short locations.
- **Protection features**: OVP/OCP/OPP protect both the supply and your board during testing.
- **Waveform display**: Monitor voltage/current stability in real time during power-up.
- **Preset groups**: Save your common RP6502 test configurations (e.g., M1: 3.3V/200mA for short test, M2: 5V/1A for normal operation).

**Important**: The RP6502 is normally powered via USB through the Pico modules. The DPS-150 is for **testing and debugging**, not normal operation. Use it to verify the board is safe before plugging in USB power.
